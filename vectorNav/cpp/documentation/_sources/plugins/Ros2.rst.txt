ROS2

Usage
=====

Getting Started
===============

Build and Setup
---------------

1. Download and Install ROS2 on your system https://www.ros.org/blog/getting-started/
2. Navigate to the ROS2 folder

.. code-block:: console
    
    cd cpp/plugins/ros2

3. Build the VNSDK ROS2 plugin Package

.. code-block:: console
    
    colcon build

4. Source this setup script to make the built packages available in your current shell session. 
To automatically source each time you open a termial window, you can source this file in your .bashrc file.

.. code-block:: console
    
    source install/setup.sh

5. Add the user to the dialout group to grant permission to access serial ports and modem devices

.. code-block:: console

		sudo usermod -a -G dialout <username>

6. Add the user to the tty group, which provides access to terminal devices

.. code-block:: console

    sudo usermod -a -G tty <username>

Listen to a Sensor
------------------

1. Connect a VectorNav sensor to a communication port.
2. Search for the name of the connected COM port. This is often "/dev/ttyUSB0".

.. code-block:: console
    
    ls /dev/tty*

3. Edit the port and baud rate values in ``vectornav.yaml`` to match your communication port and the baud rate configured on the sensor.

4. Source the setup script in the current terminal if it is not sourced in your ``.bashrc`` file. 

.. note::
      
      This needs to be sourced for each new terminal window that you open.

.. code-block:: console

      source install/setup.bash

5. Launch the VectorNav node:

.. code-block:: console

    ros2 launch vectornav vectornav.launch.py 

6. Open a new terminal window and verify the sensor is publishing data by listening to a topic:

.. code-block:: console

    ros2 topic echo /vectornav/raw/common
