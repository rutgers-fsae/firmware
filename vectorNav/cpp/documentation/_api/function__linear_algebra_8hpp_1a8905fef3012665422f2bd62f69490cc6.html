Template Function VN::Math::horzcat¶

  • Defined in File LinearAlgebra.hpp

Function Documentation¶

template<uint16_t n, uint16_t m1, uint16_t m2, typename T>
Matrix<n, m1 + m2, T> VN::Math::horzcat(const Matrix<n, m1, T> &a, const Matrix<n, m2, T> &b) noexcept¶

Concatenates two matrices horizontally.

Template Parameters:

  • n – The number of rows in both matrices.

  • m1 – The number of columns in the first matrix.

  • m2 – The number of columns in the second matrix.

  • T – The type of the elements in the matrices.

Parameters:

  • a – The first matrix.

  • b – The second matrix.

Returns:

The resulting matrix after horizontal concatenation.
