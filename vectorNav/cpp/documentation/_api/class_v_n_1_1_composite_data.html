class CompositeData¶

The standard user interface for receiving all measurement data. Auto-generated from protocol. While large in memory use by default, this can be pared down with compile-configuration settings.

struct AttitudeGroup¶

The attitude group provides all estimated outputs which are dependent upon the estimated attitude solution. The attitude will be derived from either the AHRS or the INS, depending upon which filter is currently active and tracking. All of the fields in this group will only be valid if the AHRS/INS filter is currently enabled and tracking.

struct Gnss2Group¶

The GNSS 2 group provides all outputs which are dependent upon the measurements collected from the secondary GNSS receiver. All data in this group is updated at the rate of the GPS receiver (nominally 5Hz for the internal GNSS).

struct Gnss3Group¶

The GNSS 3 group provides all outputs which are dependent upon the measurements collected from the secondary onboard GNSS, if external GNSS is enabled. All data in this group is updated at the rate of the GNSS receiver (nominally 5Hz for the internal GNSS).

struct GnssGroup¶

The GNSS group provides all outputs which are dependent upon the measurements collected from the primary GNSS receiver. All data in this group is updated at the rate of the GPS receiver (nominally 5Hz for the internal GNSS).

struct ImuGroup¶

The IMU group provides all outputs which are dependent upon the measurements collected from the IMU.

struct InsGroup¶

The INS group provides all estimated outputs which are dependent upon the onboard INS state solution. All of the fields in this group will only be valid if the INS filter is currently enabled and tracking.

struct TimeGroup¶

The time group provides all timing and event counter related outputs. Some of these outputs (such as the TimeGps, TimePps, and TimeUtc), require either that the internal GPS to be enabled, or an external GPS must be present.
