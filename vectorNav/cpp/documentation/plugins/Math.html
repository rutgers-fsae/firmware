Math¶

The Math plugin includes several classes, functions, and universal constants that are available to the user for various mathematical tasks.

Usage¶

This plugin is contained in a single Math namespace but split into multiple files based on the type of mathematical task performed:

  • Temperature and angle conversions are located in the Conversions file.

  • Conversions that are related to attitude are located in the AttitudeMath file. This includes transformations between the different attitude representations, such as the direction cosine matrix (DCM), quaternion, and yaw-pitch-roll angles. Additionally, quaternion propagation and methods for calculating attitude from inertial sensor data can be found in this file.

  • Any calculations related to position and velocity can be found in the PosVelMath file. This includes course and speed over ground calculations and the conversions between Earth-centered, Earth-fixed (ECEF) and latitude, longitude, altitude (LLA) coordinates.

  • All math related to linear algebra can be found in the LinearAlgebra file. Some helpful functions located here include the inverse, transpose, dot product, and cross product methods.

  • Linear least squares solvers and utilities are located in the LinearLeastSquares file. This includes the LinearLeastSquares class for solving systems, along with result containers and error handling.

  • Mathematical routines related to eigen decomposition can be found in the EigenDecomposition file. This includes the EigenDecomposition class for operations like extracting eigenvalues and eigenvectors, which is used in linear system solvers and other numerical routines.

Note

While not part of the Math plugin, matrix operator overloads along with the identity and null functions are provided in Matrix.
