ROS2¶

Usage¶

Getting Started¶

Build and Setup¶

  1. Download and Install ROS2 on your system https://www.ros.org/blog/getting-started/

  2. Navigate to the ROS2 folder

cd cpp/plugins/ros2

  1. Build the VNSDK ROS2 plugin Package

colcon build

4. Source this setup script to make the built packages available in your current shell session. To automatically source each time you open a termial window, you can source this file in your .bashrc file.

source install/setup.sh

  1. Add the user to the dialout group to grant permission to access serial ports and modem devices

sudo usermod -a -G dialout <username>

  1. Add the user to the tty group, which provides access to terminal devices

sudo usermod -a -G tty <username>

Listen to a Sensor¶

  1. Connect a VectorNav sensor to a communication port.

  2. Search for the name of the connected COM port. This is often “/dev/ttyUSB0”.

ls /dev/tty*

  1. Edit the port and baud rate values in vectornav.yaml to match your communication port and the baud rate configured on the sensor.

  2. Source the setup script in the current terminal if it is not sourced in your .bashrc file.

Note

This needs to be sourced for each new terminal window that you open.

source install/setup.bash

  1. Launch the VectorNav node:

ros2 launch vectornav vectornav.launch.py

  1. Open a new terminal window and verify the sensor is publishing data by listening to a topic:

ros2 topic echo /vectornav/raw/common
